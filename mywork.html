Hoppy Hour

Full Stack application built with Sequelize, Node.js, and Javascript. I was tasked with creating an app incorporating authorization as well as an API. Using the Google Places API I created a web-based app which first verifies the legal drinking age of the user, then prompts an account signup. After successfully creating an account, the app will geolocate the user and display all bars or pubs in the nearby vicinity serving beer. Users can also keep track of favorite spots, saving them to their unique account.

CodePrep | Full Stack Ruby on Rails

This is a full stack Ruby on Rails application built by myself and three other team members. Designed and developed for fellow programmers prepping for their first technical interviews, CodePrep simulates the whiteboarding experience complete with randomly generated questions, peer review, and a countdown clock feature. As the project manager for this project, I was responsible for delegating tasks and helping troubleshoot any bugs or technical constraints given the deadline. More specifically, I also worked on the front end as well as seeding the database.
